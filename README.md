# Customer Churn Analysis

## Description
This project analyzes customer churn in a telecom company, leveraging both Python and SQL to derive insights and predict risk levels.

## Objectives
- Understand factors contributing to customer churn.
- Develop a churn risk scoring system.

## Technologies Used
- Python (Pandas, NumPy)
- SQL
- Jupyter Notebook

## Data Source
- [https://www.kaggle.com/datasets/blastchar/telco-customer-churn]
